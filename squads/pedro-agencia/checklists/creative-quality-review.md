# Checklist: Creative Quality Review

## Como usar

Execute este checklist antes de aprovar qualquer output criativo — logos, layouts, vídeos, animações, peças publicitárias, apresentações visuais ou qualquer material que carregue a identidade visual da marca.
**O pa-genesis é o guardião final da qualidade criativa.** Nenhum asset sai sem revisão completa.

**Princípio:** Criatividade sem disciplina é caos. Disciplina sem criatividade é tédio. Este checklist garante o equilíbrio.

---

## 🔴 SEÇÃO 1: Concept & Strategy Alignment

### Aderência ao Brief
- [ ] O conceito criativo responde diretamente ao brief recebido
- [ ] Single-Minded Proposition (SMP) claramente comunicada na peça
- [ ] Audiência-alvo identificável como destinatária da comunicação
- [ ] Tom e manner conforme especificado no creative brief
- [ ] Objetivos de campanha atendidos pela abordagem criativa
- [ ] Não há desvio estratégico entre brief e execução

### Originalidade & Impacto
- [ ] A ideia é genuinamente original — não cópia ou variação óbvia de campanha existente
- [ ] O conceito gera reação emocional (surpresa, curiosidade, identificação, desejo)
- [ ] Diferencia-se claramente das abordagens de concorrentes
- [ ] A ideia é adaptável a múltiplos formatos e canais
- [ ] O conceito é memorável — passaria no "teste do bar" (alguém lembraria e contaria)

### Teste de Viabilidade
- [ ] Conceito é executável dentro do budget disponível
- [ ] Prazo de produção é realista para o conceito proposto
- [ ] Recursos técnicos necessários estão disponíveis
- [ ] Restrições legais e regulatórias são compatíveis com a ideia

---

## 🔴 SEÇÃO 2: Visual Execution

### Composição & Hierarquia
- [ ] Hierarquia visual clara — o olho sabe para onde ir primeiro
- [ ] Ponto focal definido e intencional
- [ ] Elementos organizados com lógica (grid, alinhamento, proximidade)
- [ ] Espaço em branco (white space) utilizado de forma estratégica — sem poluição visual
- [ ] Equilíbrio visual entre elementos (simétrico ou assimétrico intencional)
- [ ] Regra dos terços aplicada quando relevante
- [ ] Z-index e sobreposições intencionais e funcionais

### Tipografia
- [ ] Fontes aprovadas pelo brandbook utilizadas
- [ ] Máximo de 2-3 famílias tipográficas por peça
- [ ] Hierarquia tipográfica clara (headline > subheadline > corpo > legal)
- [ ] Tamanho mínimo de leitura respeitado (mínimo 12px para corpo em digital)
- [ ] Contraste texto/fundo suficiente para legibilidade (WCAG AA minimum 4.5:1)
- [ ] Kerning, tracking e leading verificados visualmente
- [ ] Nenhuma viúva ou órfã em textos longos
- [ ] Fontes com licença de uso comercial verificada

### Cores
- [ ] Paleta de cores do brandbook respeitada (códigos HEX/RGB/CMYK exatos)
- [ ] Contraste adequado para acessibilidade
- [ ] Cores funcionam em modo escuro (quando aplicável)
- [ ] Gradientes suaves e intencionais — sem banding
- [ ] Cor primária da marca dominante visualmente
- [ ] Cores de CTA contrastam com o fundo e chamam atenção

### Imagens & Fotografia
- [ ] Resolução adequada ao meio final (72dpi web, 300dpi impresso)
- [ ] Imagens autênticas — sem aparência genérica de stock óbvio
- [ ] Diversidade e representatividade consideradas
- [ ] Direitos de uso confirmados (licença, produção própria ou banco contratado)
- [ ] Tratamento de cor consistente entre todas as imagens da campanha
- [ ] Recortes e máscaras limpos — sem serrilhados ou halos
- [ ] Proporção de imagem correta para o placement

---

## 🔴 SEÇÃO 3: Copy Integration

### Texto no Design
- [ ] Headline/título é a primeira coisa que se lê
- [ ] Copy integrada ao design — não parece "colada por cima"
- [ ] Texto não compete visualmente com a imagem principal
- [ ] CTA visível, claro e com contraste suficiente
- [ ] Quantidade de texto proporcional ao tempo de atenção disponível no formato
- [ ] Nenhuma palavra cortada, truncada ou ilegível
- [ ] Textos em idiomas corretos (verificar localização se aplicável)

### Legibilidade
- [ ] Texto legível em tamanho mobile (testar em tela de 5-6 polegadas)
- [ ] Texto legível em desktop (testar em 13-15 polegadas)
- [ ] Contraste de cor texto/fundo passa em verificador WCAG
- [ ] Textos sobre imagens têm overlay ou sombra para garantir leitura
- [ ] Legendas e disclaimers legíveis (mesmo em tamanho reduzido)

---

## 🔴 SEÇÃO 4: Video & Motion (quando aplicável)

### Qualidade Técnica
- [ ] Resolução mínima 1080p (4K quando especificado)
- [ ] Frame rate correto para a plataforma (24, 25, 30 ou 60fps)
- [ ] Bitrate adequado para qualidade sem compressão excessiva
- [ ] Codec correto (H.264 para web, ProRes para edição)
- [ ] Áudio sincronizado com vídeo (sem lip-sync issues)
- [ ] Sem flickering, tearing ou artifacts visuais
- [ ] Transições suaves e intencionais

### Narrativa Visual
- [ ] Gancho nos primeiros 3 segundos (hook)
- [ ] Storytelling claro: início, meio e fim
- [ ] Ritmo adequado à plataforma e audiência
- [ ] Legendas corretas e sincronizadas (obrigatório para consumo sem áudio)
- [ ] Thumbnail/frame de capa intencional e atrativa
- [ ] Duração conforme especificação da plataforma
- [ ] End frame com CTA e branding claros

### Áudio
- [ ] Trilha sonora com licença verificada e documentada
- [ ] Nível de áudio balanceado (música não cobre locução)
- [ ] Locução profissional e clara
- [ ] Sem ruídos de fundo, pops ou distorções
- [ ] Mixagem final masterizada para plataforma alvo

---

## 🟡 SEÇÃO 5: Platform & Format Compliance

### Especificações por Plataforma
- [ ] Dimensões exatas conforme specs de cada plataforma (ver guia de specs do squad)
- [ ] Peso do arquivo dentro dos limites (Facebook 30MB imagem, TikTok 4GB vídeo, etc.)
- [ ] Safe zones respeitadas (stories com margens para UI elements)
- [ ] Formato de arquivo correto (JPG, PNG, MP4, GIF, SVG conforme uso)
- [ ] Proporção mantida sem distorção ou crop inesperado
- [ ] Texto em área segura de stories/reels (sem sobreposição de elementos da plataforma)

### Versões e Adaptações
- [ ] Todas as variações de tamanho produzidas conforme plano de mídia
- [ ] Adaptações não são apenas redimensionamentos — conceito adaptado ao formato
- [ ] Versão horizontal, vertical e quadrada produzidas quando necessário
- [ ] Dark mode version testada (quando aplicável)
- [ ] Impresso vs. digital: cores convertidas corretamente (RGB → CMYK)

---

## 🔴 SEÇÃO 6: Brand Consistency

### Identidade Visual
- [ ] Logo na versão, posição e tamanho corretos
- [ ] Logo não distorcido, cortado ou sobreposto indevidamente
- [ ] Área de respeito do logo respeitada
- [ ] Tagline presente quando exigida pelo brandbook
- [ ] Elementos visuais proprietários utilizados corretamente (ícones, patterns, ilustrações)
- [ ] Estilo de fotografia consistente com diretrizes da marca
- [ ] Treatment de imagem (filters, overlays) alinhado ao brand style

### Voz e Personalidade
- [ ] Tom visual comunica a personalidade da marca (ex.: sofisticado, acessível, ousado)
- [ ] Estilo de copy consistente com Brand Voice Guide
- [ ] Elementos visuais reforçam o posicionamento da marca
- [ ] Nenhuma associação visual indesejada (cores ou formas que remetam a concorrentes)

---

## 🔴 SEÇÃO 7: Accessibility & Inclusão

### Acessibilidade Visual
- [ ] Contraste mínimo WCAG AA (4.5:1 para texto normal, 3:1 para grande)
- [ ] Texto não depende apenas de cor para transmitir significado
- [ ] Alt text preparado para todas as imagens (descrição funcional)
- [ ] Legendas abertas ou fechadas em todos os vídeos
- [ ] Animações com opção de redução de movimento (prefers-reduced-motion)

### Inclusão
- [ ] Representatividade considerada em fotos e ilustrações
- [ ] Linguagem visual inclusiva e não estereotipada
- [ ] Evitar ableism visual ou representações preconceituosas
- [ ] Diversidade de corpos, idades, etnias e gêneros quando retrata pessoas
