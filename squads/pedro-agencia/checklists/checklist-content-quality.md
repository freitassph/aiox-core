# Checklist: Content Quality

## Como usar

Execute este checklist antes de qualquer publicação de conteúdo — posts, artigos, vídeos, e-mails, anúncios, whitepapers, ou qualquer outro formato.
**Nenhum conteúdo vai ao ar sem revisão completa.** Qualidade não é negociável.

O responsável por cada seção está indicado. O pa-chronicle valida o checklist completo antes da aprovação final.

---

## 🔴 SEÇÃO 1: Copy Quality — A palavra certa no lugar certo

### Ortografia & Gramática
- [ ] Texto revisado em ferramenta de ortografia (Grammarly, LanguageTool ou similar)
- [ ] Revisão humana feita por alguém além do autor
- [ ] Acentuação correta em todo o texto
- [ ] Crase usada corretamente
- [ ] Concordância verbal e nominal verificada
- [ ] Pontuação revisada (vírgulas, pontos, dois-pontos, ponto-e-vírgula)
- [ ] Parágrafos com comprimento adequado (máx. 4-5 linhas para digital)
- [ ] Frases sem ambiguidades que possam confundir o leitor
- [ ] Repetição excessiva de palavras verificada e corrigida
- [ ] Jargão desnecessário eliminado ou explicado

### Clareza & Objetividade
- [ ] A mensagem principal está clara nos primeiros 2 parágrafos
- [ ] O texto responde as perguntas: O quê? Para quem? Por quê?
- [ ] Linguagem adequada ao nível de conhecimento do público-alvo
- [ ] Termos técnicos explicados quando necessário
- [ ] Call-to-action (CTA) claro, específico e único
- [ ] Próximo passo para o leitor está evidente
- [ ] Benefícios destacados antes de características
- [ ] Frases ativas preferencialmente ao invés de passivas
- [ ] Analogias e exemplos usados para clarificar conceitos complexos
- [ ] Texto escaneável com uso de subtítulos, listas e negrito estratégico

### Persuasão & Engajamento
- [ ] Headline/título é irresistível e promete valor real
- [ ] Lead (abertura) prende a atenção nos primeiros 10 segundos
- [ ] Storytelling presente quando adequado ao formato
- [ ] Provas sociais incluídas (dados, depoimentos, cases)
- [ ] Senso de urgência ou escassez (quando verdadeiro e relevante)
- [ ] Objeções antecipadas e respondidas no texto
- [ ] Tom conversacional mantido (fala com o leitor, não para o leitor)
- [ ] Personalização presente (pronomes de 2ª pessoa, contexto do leitor)

## 🔴 SEÇÃO 2: SEO — Encontrável e relevante

### On-Page Fundamentals
- [ ] Keyword principal definida e validada em ferramenta de SEO
- [ ] Keyword principal presente no H1/título
- [ ] Keyword principal nos primeiros 100 palavras do texto
- [ ] Keyword principal na meta description
- [ ] Keywords secundárias (2-3) distribuídas naturalmente no texto
- [ ] Densidade de keyword adequada (1-2%, sem keyword stuffing)
- [ ] URL amigável, curta e com keyword principal
- [ ] Alt text em todas as imagens (descritivo e com keyword quando natural)
- [ ] H2s e H3s usados corretamente com keywords relacionadas
- [ ] Internal links para conteúdos relevantes do próprio site
- [ ] External links para fontes autoritativas (quando aplicável)
- [ ] Meta title entre 50-60 caracteres
- [ ] Meta description entre 150-160 caracteres com CTA

### Estrutura & UX de Busca
- [ ] Conteúdo responde a search intent (informacional, navegacional, transacional)
- [ ] Featured snippet opportunity explorada (listas, tabelas, definições)
- [ ] Schema markup implementado quando aplicável
- [ ] Tempo de leitura estimado compatível com profundidade do tema
- [ ] Conteúdo mais completo que os top 3 resultados para a keyword
- [ ] FAQ section incluída quando keywords de pergunta são relevantes
- [ ] Título do artigo diferente do title tag (quando vantajoso)

## 🔴 SEÇÃO 3: Brand Voice — Sempre reconhecível

### Consistência de Voz
- [ ] Tom da marca mantido (consultar Brand Voice Guide)
- [ ] Personalidade da marca refletida (consultivo, inspirador, direto, etc.)
- [ ] Vocabulário da marca utilizado (termos proprietários, gírias da marca)
- [ ] Expressões proibidas pela marca ausentes
- [ ] Nível de formalidade correto para o canal e audiência
- [ ] Humor e leveza aplicados apenas onde a marca permite
- [ ] Posicionamento da marca reforçado implicitamente

### Alinhamento Estratégico
- [ ] Conteúdo alinhado ao pilar estratégico definido no planejamento
- [ ] Mensagem consistente com outras peças da campanha
- [ ] Posicionamento competitivo correto (não denigre concorrentes, destaca diferenciais)
- [ ] Promessa de marca respeitada e reforçada
- [ ] Tagline ou elementos da campanha presentes quando aplicável

---

## 🔴 SEÇÃO 4: Visual Quality — O que os olhos veem importa

### Imagens & Fotos
- [ ] Resolução adequada para o meio (72dpi web, 300dpi impresso)
- [ ] Proporção correta para o placement (ver especificações por plataforma)
- [ ] Imagens sem pixelização ou compressão excessiva
- [ ] Fotos com pessoas: expressões naturais e genuínas
- [ ] Diversidade e representatividade consideradas
- [ ] Direitos de uso das imagens confirmados (licença ou produção própria)
- [ ] Watermarks de bancos de imagens removidos
- [ ] Paleta de cores alinhada ao brandbook
- [ ] Contraste adequado para leitura de textos sobrepostos
- [ ] Imagem principal comunica a mensagem mesmo sem texto

### Design & Layout
- [ ] Hierarquia visual clara (o olho sabe onde ir primeiro)
- [ ] Espaço em branco (white space) adequado — sem visual poluído
- [ ] Tipografia correta e consistente com a identidade visual
- [ ] Tamanho de fonte legível em mobile (mínimo 14px para corpo)
- [ ] Logo na posição e versão corretas (positivo/negativo, tamanho mínimo)
- [ ] Grid e alinhamentos consistentes
- [ ] Cores exatas do brandbook (códigos hex/RGB/CMYK verificados)
- [ ] Gradientes e efeitos alinhados ao estilo visual da marca
- [ ] Ícones consistentes em estilo (outline vs filled, família tipográfica)
- [ ] Animações (se houver) com velocidade e easing corretos

### Vídeo (quando aplicável)
- [ ] Qualidade de imagem mínima 1080p
- [ ] Áudio limpo sem ruídos de fundo
- [ ] Legendas presentes e corretas (obrigatório para acessibilidade)
- [ ] Abertura impactante nos primeiros 3 segundos
- [ ] Duração adequada para o formato e plataforma
- [ ] Formato exportado conforme spec da plataforma
- [ ] Thumbnail personalizada e atrativa
- [ ] Trilha sonora/música com licença verificada

---

## 🟡 SEÇÃO 5: Factual Accuracy — Só a verdade resiste

### Verificação de Dados
- [ ] Todos os dados e estatísticas têm fonte identificada
- [ ] Fontes são recentes (preferencialmente < 2 anos)
- [ ] Fontes são autoritativas e confiáveis
- [ ] Links para fontes funcionam e apontam para o conteúdo correto
- [ ] Números e percentuais foram verificados manualmente
- [ ] Afirmações sobre produtos/serviços do cliente verificadas com o cliente
- [ ] Afirmações sobre concorrentes são factuais e verificáveis
- [ ] Estudos de caso têm permissão de uso do cliente referenciado
- [ ] Citações estão entre aspas e atribuídas corretamente

### Atualidade
- [ ] Informações não estão desatualizadas (verificado data de publicação das fontes)
- [ ] Referências a eventos/tendências são atuais e relevantes
- [ ] Preços e disponibilidades verificados (para e-commerce)
- [ ] Links externos ainda funcionam e apontam para conteúdo atual
- [ ] Nenhuma referência a conteúdo arquivado ou descontinuado

---

## 🔴 SEÇÃO 6: Legal & Compliance — Sem surpresas jurídicas

### Publicidade & Regulatório
- [ ] CONAR compliance verificado (para publicidade)
- [ ] Regulamentações específicas do setor verificadas (saúde, financeiro, educação)
- [ ] Superlativos e afirmações absolutas comprovados ou removidos ("o melhor", "o único")
- [ ] Disclaimers necessários presentes e legíveis
- [ ] Asteriscos e letras miúdas corretos e completos
- [ ] Promoções com regulamento publicado e referenciado
- [ ] Sorteios/concursos com registro no órgão competente
- [ ] Depoimentos são de pessoas reais e verificáveis

### Privacidade & Dados
- [ ] LGPD compliance verificado para coleta de dados
- [ ] Política de privacidade linkada quando há coleta de dados
- [ ] Opt-in explícito para e-mail marketing
- [ ] Dados de terceiros usados com consentimento documentado
- [ ] Nenhum dado pessoal sensível exposto inadvertidamente

### Propriedade Intelectual
- [ ] Direitos de música e áudio verificados (ECAD/licença)
- [ ] Fontes tipográficas com licença para uso comercial
- [ ] Imagens com licença adequada ao uso (web, impresso, comercial)
- [ ] Nenhum elemento protegido por marca registrada de terceiros
- [ ] Referências a marcas de terceiros factualmente corretas

---

## 🔴 SEÇÃO 7: Platform Specs — O formato certo no lugar certo

### Digital/Web
- [ ] Imagens otimizadas para web (< 200KB para imagens de suporte)
- [ ] Imagens hero/banner < 500KB
- [ ] Lazy loading implementado para páginas com muitas imagens
- [ ] Texto alternativo em todas as imagens
- [ ] Vídeos em formato HTML5 ou embed otimizado
- [ ] Links com target="_blank" para externos e rel="noopener"

### Redes Sociais
- [ ] Dimensões verificadas conforme plataforma (ver guia de specs)
- [ ] Limite de caracteres respeitado (Twitter 280, LinkedIn 3000, etc.)
- [ ] Hashtags verificadas — relevantes, sem conotação negativa
- [ ] Menções (@) testadas e para contas corretas
- [ ] Preview de link (OG tags) verificado antes de publicar
- [ ] Stories: texto em safe zone (sem corte nos bordos)
- [ ] Emojis testados em diferentes sistemas operacionais

### E-mail
- [ ] Linha de assunto < 50 caracteres (idealmente)
- [ ] Preheader text configurado
- [ ] Renderização testada em Gmail, Outlook, Apple Mail
- [ ] Visualização mobile testada
- [ ] Imagens com fallback de texto (alt text)
- [ ] Unsubscribe link presente e funcional
- [ ] Endereço físico presente (CAN-SPAM / LGPD)
- [ ] Largura máxima 600px para e-mail HTML
