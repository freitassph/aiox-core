## Technical SEO Checklist

### Core Web Vitals
| Metric | Good | Needs Improvement | Poor |
|--------|------|-------------------|------|
| LCP (Largest Contentful Paint) | < 2.5s | 2.5-4.0s | > 4.0s |
| INP (Interaction to Next Paint) | < 200ms | 200-500ms | > 500ms |
| CLS (Cumulative Layout Shift) | < 0.1 | 0.1-0.25 | > 0.25 |

### Technical Audit Items
- [ ] HTTPS enabled and properly configured
- [ ] Mobile-friendly (responsive design)
- [ ] XML sitemap exists and submitted to GSC
- [ ] Robots.txt doesn't block important pages
- [ ] No index on thin/duplicate content pages
- [ ] Canonical tags on all pages
- [ ] Structured data (Schema.org) implemented
- [ ] 404 page is helpful (not generic)
- [ ] 301 redirects for all broken URLs
- [ ] Site speed under 3 seconds (desktop)
- [ ] Site speed under 4 seconds (mobile)
- [ ] No more than 3 redirect chains
- [ ] Hreflang tags for multi-language sites
- [ ] Breadcrumbs implemented
- [ ] Internal linking structure logical
- [ ] URL structure is clean and descriptive

## On-Page SEO Checklist

### Per Page
- [ ] Title tag: 50-60 characters, keyword near front
- [ ] Meta description: 150-160 characters, compelling CTA
- [ ] H1: One per page, contains target keyword
- [ ] H2-H6: Logical hierarchy, keyword variations
- [ ] Keyword density: 1-2% (natural, not forced)
- [ ] Image ALT text: Descriptive, keyword-relevant
- [ ] Internal links: 3-5 to related content
- [ ] External links: 2-3 to authoritative sources
- [ ] URL slug: Short, descriptive, keyword-rich
- [ ] Content length: 1,500-2,500 words (ranking content)
- [ ] Readability: Flesch score 60+ (accessible)
- [ ] Media: Images, videos, infographics included
- [ ] Schema markup: Article, FAQ, HowTo as relevant

### Content Quality Signals
- [ ] E-E-A-T demonstrated (Experience, Expertise, Authoritativeness, Trust)
- [ ] Original research or data cited
- [ ] Author bio with credentials
- [ ] Last updated date visible
- [ ] Sources cited and linked
- [ ] Comprehensive coverage of topic
- [ ] User intent satisfied (informational, navigational, transactional)

## Off-Page SEO Strategy

### Link Building Tactics
| Tactic | Difficulty | Impact | Timeline |
|--------|-----------|--------|----------|
| Guest posting | Medium | High | 2-4 weeks |
| Broken link building | Medium | Medium | 2-6 weeks |
| Resource page outreach | Easy | Medium | 1-3 weeks |
| HARO (Help a Reporter Out) | Easy | High | Ongoing |
| Digital PR | Hard | Very High | 4-8 weeks |
| Skyscraper content | Medium | High | 4-6 weeks |
| Original research/studies | Hard | Very High | 6-12 weeks |
| Testimonials for tools | Easy | Low | 1-2 weeks |

### Link Quality Metrics
- Domain Authority (DA): Target 30+ for quality links
- Relevance: Links from same/niche industry preferred
- Traffic: Referring domain has organic traffic
- Anchor text: Natural, varied, brand-heavy
- Follow vs. NoFollow: Aim for 70/30 ratio

## Keyword Research Framework

### Keyword Intent Types
| Intent | Description | Example | Content Type |
|--------|-------------|---------|-------------|
| Informational | Seeking knowledge | "how to do seo" | Guide, blog post |
| Navigational | Seeking specific site | "hubspot login" | Brand page |
| Commercial | Researching before buying | "best seo tools" | Comparison, review |
| Transactional | Ready to buy | "buy ahrefs subscription" | Product page |

### Keyword Prioritization Matrix
```
Priority Score = (Search Volume × CTR Estimate × Business Value) / Keyword Difficulty

Where:
- Search Volume: Monthly searches
- CTR Estimate: Based on current position potential
- Business Value: 1-10 based on conversion likelihood
- Keyword Difficulty: 0-100 (Ahrefs/Semrush)
```

### Keyword Mapping
1. **Pillar Pages:** Broad, high-volume keywords (1,000+ searches/month)
2. **Cluster Content:** Long-tail, specific keywords (100-1,000 searches/month)
3. **Supporting Content:** Very long-tail, question-based (10-100 searches/month)

## Content Optimization Checklist

### Pre-Publication
- [ ] Target keyword identified
- [ ] Search intent matched (analyze top 10 results)
- [ ] Keyword in title, H1, first 100 words
- [ ] 3-5 related keywords included naturally
- [ ] Meta title and description written
- [ ] URL slug optimized
- [ ] Images optimized (compressed, ALT text)
- [ ] Internal links to 3-5 relevant pages
- [ ] External links to 2-3 authoritative sources
- [ ] Schema markup added (if applicable)
- [ ] Readability checked (Grammarly, Hemingway)
- [ ] Word count competitive with top 10 results

### Post-Publication
- [ ] Submitted to Google for indexing (GSC)
- [ ] Shared on social media channels
- [ ] Linked from existing relevant pages
- [ ] Added to sitemap
- [ ] Performance tracked in analytics

### Monitoring (Monthly)
- [ ] Ranking position tracked
- [ ] Organic traffic monitored
- [ ] Click-through rate analyzed
- [ ] Backlinks acquired tracked
- [ ] Content updated if ranking drops

## SEO Reporting Template

### Monthly Report Structure
1. **Executive Summary:** Key wins, challenges, next steps
2. **Organic Traffic:** Sessions, users, pageviews (MoM, YoY)
3. **Keyword Rankings:** Top 20, movers, new entries
4. **Backlinks:** New links, lost links, top referring domains
5. **Technical Health:** Core Web Vitals, crawl errors, index coverage
6. **Content Performance:** Top pages, engagement metrics
7. **Competitor Analysis:** Share of voice, gap analysis
8. **Next Month Plan:** Prioritized action items
