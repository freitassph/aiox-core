## Visão Geral (Overview)

### Directory Structure

Este squad contém a estrutura completa de agents, tasks, workflows, checklists e templates para operar uma agência de marketing digital Tier S+++. Cada directory está organizado para facilitar navegação e automação.

```
pedro-agencia/
├── agents/          # 22 agentes especializados
├── tasks/           # 36 tasks com entrada/saida/checklist
├── workflows/       # 7 workflows multi-step com retry logic

├── templates/       # 4 templates reutilizáveis
├── config/          # Configurações e standards

└── scripts/         # Automation scripts
```

## Como Usar o Squad

### Ativação Direta
```bash
# Ativar o orquestrador principal
@pa-empire

# Ativar especialistas
@pa-oracle      # Brand Strategy
@pa-quill       # Copywriting
@pa-vertex      # SEO
@pa-inbox       # Email Marketing
@pa-profit      # Performance Marketing
# ... qualquer dos 22 agentes
```

### Slash Commands
```bash
/pedro-agencia:agents:pa-empire
/pedro-agencia:agents:pa-oracle
/pedro-agencia:agents:pa-genesis
```

### Workflows Completos
```bash
# Campaign completa (da estratégia ao ar)
*pa-launch-campaign --client "..." --product "..." --budget "..."

# Brand building (identidade até distribuição)
*pa-brand-build --client "..." --market "..."

# Growth sprint (30/60/90 dias)
*pa-growth-sprint --client "..." --duration 30d|60d|90d

# Content engine (produção em escala)
*pa-content-engine --client "..." --frequency "..."
```

## Filosofia do Squad

> "Uma agência de $70B não vende serviços — vende transformações. Cada agente do Pedro Agência Squad opera com a mentalidade de que o sucesso do cliente é o único resultado que importa."

### Princípios Inegociáveis:
1. **Excelência sem desculpas** — Tier S+++ não aceita mediocridade
2. **DNA de Elite** — cada agente carrega o conhecimento dos melhores da história
3. **Dados antes de intuição** — toda decisão tem fundamento analítico
4. **Integração real** — os agentes trabalham em conjunto, não em silos
5. **Resultados mensuráveis** — ROI é o idioma comum de todos os agentes

## Critérios de Qualidade Global

Antes de qualquer entrega ao cliente, o squad verifica:
- [ ] Estratégia alinhada com objetivos de negócio do cliente?
- [ ] Todos os deliverables passaram pelo checklist do agente responsável?
- [ ] Integração entre agentes garantida (pa-oracle → pa-genesis → pa-quill)?
- [ ] Métricas de sucesso definidas e trackadas?
- [ ] pa-prism com dashboard de acompanhamento configurado?
- [ ] pa-anchor com onboarding do cliente documentado?
