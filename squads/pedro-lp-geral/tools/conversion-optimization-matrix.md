## Conversion Leak Detection

### The 5-Second Test
A visitor should understand within 5 seconds:
1. **What** you offer
2. **Who** it's for
3. **Why** they should care
4. **What** to do next

| Question | Pass Criteria | Check |
|----------|--------------|-------|
| Can you state the offer in one sentence? | Clear value proposition above the fold | [ ] |
| Is the target audience obvious? | "For [specific audience]" visible immediately | [ ] |
| Is the primary benefit clear? | Main benefit in headline or subheadline | [ ] |
| Is there exactly ONE primary CTA? | Single, visually dominant action button | [ ] |

## Headline Optimization

### Headline Scoring (0-10)

| Criteria | Points | Description |
|----------|--------|-------------|
| Specific | 0-2 | Contains numbers, data, or specific claims |
| Relevant | 0-2 | Directly addresses target audience pain point |
| Emotional | 0-2 | Triggers emotion (curiosity, fear, desire, urgency) |
| Clear | 0-2 | No jargon, instantly understandable |
| Benefit-driven | 0-2 | Focuses on outcome, not features |

**Score 8-10:** Excellent — test minor variations
**Score 5-7:** Good — test against stronger alternative
**Score 0-4:** Weak — rewrite immediately

### Headline Formulas That Convert

| Formula | Example |
|---------|---------|
| How to [desired outcome] in [timeframe] | "How to Double Your Revenue in 90 Days" |
| The [number] [adjective] Ways to [benefit] | "7 Proven Ways to Reduce Customer Churn" |
| [Result] Without [pain point] | "Grow Your Team Without the Hiring Headache" |
| Why [common belief] Is Wrong (And What Works) | "Why Cold Calling Is Dead (And What Works)" |
| The Ultimate Guide to [topic] | "The Ultimate Guide to Email Marketing" |

## CTA Optimization

### CTA Best Practices

| Element | Best Practice | Check |
|---------|--------------|-------|
| Text | Action-oriented verb + benefit | [ ] "Start My Free Trial" not "Submit" |
| Color | Contrasts with page (stands out) | [ ] |
| Size | Large enough to be unmistakable | [ ] |
| Position | Above the fold + repeated after key sections | [ ] |
| Urgency | Time-bound or scarcity element | [ ] |
| Risk reversal | "Free", "No credit card", "Cancel anytime" | [ ] |

### CTA Copy Scorecard

| Copy | Conversion Potential | Why |
|------|---------------------|-----|
| "Submit" | Very Low | Generic, feels like work |
| "Click Here" | Very Low | Vague, no value signal |
| "Learn More" | Low | Low commitment but low intent |
| "Get Started" | Medium | Clear action but no benefit |
| "Start Free Trial" | High | Action + benefit + risk reversal implied |
| "Get My Free Audit" | Very High | Personal + benefit + zero risk |
| "Claim My [Result]" | Very High | Ownership + specific benefit |

## Social Proof Hierarchy

| Type | Strength | When to Use |
|------|----------|-------------|
| Case study with data | Highest | B2B, high-ticket offers |
| Video testimonial | Very High | Any — builds trust fast |
| Written testimonial with photo | High | Standard landing pages |
| Logo bar (known brands) | High | B2B credibility |
| Number-based proof | Medium-High | "10,000+ customers served" |
| Star ratings | Medium | E-commerce, local services |
| Media mentions | Medium | "As seen in..." |
| User count | Medium | Network effect businesses |

## Urgency & Scarcity Framework

| Type | Example | Authenticity Required |
|------|---------|----------------------|
| Time-limited | "Offer ends Friday" | MUST be real deadline |
| Quantity-limited | "Only 3 spots left" | MUST be actual count |
| Exclusive | "Invite-only access" | MUST have actual exclusivity |
| Seasonal | "Back to school pricing" | Tied to real calendar event |
| Launch | "Early bird: 48h left" | MUST honor deadline |

**WARNING:** Fake urgency destroys trust and long-term conversion rates.

## Objection Handling Matrix

| Common Objection | Where to Address | How |
|-----------------|------------------|-----|
| "Too expensive" | Near pricing/CTA | ROI calculator, payment plans, comparison |
| "Will it work for me?" | Mid-page | Case studies by segment, guarantees |
| "I need to think about it" | After key benefit | Risk reversal, limited-time bonus |
| "I don't have time" | Early | "Set up in 5 minutes", "Done for you" |
| "Is this legit?" | Above the fold | Logos, certifications, media mentions |
| "What if it doesn't work?" | Near CTA | Guarantee, refund policy, testimonials |
