# Industry Adaptation Guide

> **Universal landing page adaptation guide for any industry or niche** — Used by lp-geral-orchestrator to quickly configure the Prestige Universal Design System for specific markets.

---

## Industry Profiles

### 1. SaaS / Technology

| Element | Adaptation |
|---------|-----------|
| **Headline tone** | Innovation-focused, metrics-driven |
| **Color palette** | Blues, purples, clean whites (trust + tech) |
| **Social proof** | Logos of companies using it, user count |
| **Key sections** | Features, integrations, pricing, demo CTA |
| **Compliance** | LGPD, SOC 2, ISO 27001 badges |
| **CTA** | "Start Free Trial", "Book a Demo" |
| **Urgency** | Free trial expiration, limited beta |

### 2. Healthcare / Medical

| Element | Adaptation |
|---------|-----------|
| **Headline tone** | Trust, care, expertise, outcomes |
| **Color palette** | Greens, teals, whites (health + calm) |
| **Social proof** | Doctor credentials, patient testimonials |
| **Key sections** | Specialties, team, insurance, booking |
| **Compliance** | CFM, LGPD (health data), ANVISA if applicable |
| **CTA** | "Agendar Consulta", "Fale Conosco" |
| **Urgency** | Appointment availability, new patient slots |

### 3. E-commerce / Retail

| Element | Adaptation |
|---------|-----------|
| **Headline tone** | Desire-driven, lifestyle-focused |
| **Color palette** | Brand colors, high contrast for CTAs |
| **Social proof** | Reviews, ratings, user-generated content |
| **Key sections** | Products, promotions, shipping, returns |
| **Compliance** | CDC (Consumer Defense Code), LGPD, payment security |
| **CTA** | "Shop Now", "Get 20% Off" |
| **Urgency** | Flash sales, limited stock, countdown timers |

### 4. Education / Infoproducts

| Element | Adaptation |
|---------|-----------|
| **Headline tone** | Transformation, results, possibility |
| **Color palette** | Warm, energetic (orange, gold, deep blue) |
| **Social proof** | Student results, income claims (verified) |
| **Key sections** | Curriculum, instructor, bonuses, guarantee |
| **Compliance** | LGPD, consumer rights, clear refund policy |
| **CTA** | "Enroll Now", "Join the Class" |
| **Urgency** | Enrollment closing, price increase, cohort limit |

### 5. Professional Services (Consulting, Legal, Accounting)

| Element | Adaptation |
|---------|-----------|

| **Social proof** | Client logos, case studies, years of experience |
| **Key sections** | Services, approach, team, contact |
| **Compliance** | Professional council rules (OAB, CRC), LGPD |
| **CTA** | "Schedule Consultation", "Get a Quote" |

### 6. Fitness / Wellness

| Element | Adaptation |
|---------|-----------|
| **Headline tone** | Aspirational, energetic, personal |
| **Color palette** | Vibrant (red, orange, green) or premium (black, gold) |

| **Key sections** | Programs, trainers, pricing, schedule |
| **Compliance** | LGPD, health disclaimers, CREFITO if applicable |

### 7. Real Estate

| Element | Adaptation |

### 8. Food & Restaurant

| Element | Adaptation |
|---------|-----------|

## Adaptation Checklist

For each new industry vertical:

- [ ] Update headline tone and vocabulary
- [ ] Configure color palette for industry expectations
- [ ] Select appropriate social proof type
- [ ] Identify compliance requirements
- [ ] Choose primary CTA language
- [ ] Define urgency/scarcity mechanism
- [ ] Map industry-specific sections
- [ ] Adjust copy frameworks (AIDA, PAS, FAB) for audience
