# Tech Stack — pedro-lp-geral

## Core
- **Framework:** Next.js 15+ (App Router, Server Components)
- **Language:** TypeScript (strict: true)
- **Styling:** Tailwind CSS v4 + CSS Custom Properties (design tokens)
- **Animation:** Framer Motion + GSAP 3 (ScrollTrigger)
- **React:** 19+

## Performance
- **Images:** next/image (WebP/AVIF, lazy load, priority)
- **Fonts:** next/font (zero layout shift, Google Fonts)
- **Bundling:** Turbopack (default Next.js 15)
- **Deploy:** Vercel (zero-config, edge network)

## Tracking & Analytics
- **Tag Manager:** Google Tag Manager (container base)
- **Analytics:** GA4
- **Ads:** Meta Pixel, Google Ads Conversion
- **Heatmaps:** Microsoft Clarity (gratuito, LGPD-friendly)
- **LGPD CMP:** CookieYes ou Cookiebot

## Forms & CRM
- **Forms:** React Hook Form + Zod validation
- **Mutation:** Next.js Server Actions
- **CRM:** HubSpot / RD Station / ActiveCampaign / Mailchimp (conforme cliente)

## Scheduling & Communication
- **Agendamento:** Cal.com (open source) ou Calendly embed
- **WhatsApp:** link wa.me com UTM + mensagem pré-preenchida

## Monitoring
- **Uptime:** UptimeRobot (gratuito)
- **Performance:** Vercel Analytics + Google Search Console
- **Error tracking:** Vercel Logs (básico) ou Sentry (se solicitado)

## Development
- **Linting:** ESLint + Prettier
- **Git:** GitHub + Vercel Git Integration (auto-deploy)
- **Node:** 20.x LTS
