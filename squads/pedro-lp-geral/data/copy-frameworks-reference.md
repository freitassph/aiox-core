# Copy Frameworks Reference — World-Class Copywriting

## Eugene Schwartz — 5 Awareness Levels

| Nível | Estado do Prospect | Estratégia de Copy |
|-------|-------------------|-------------------|
| **Unaware** | Não sabe que tem o problema | Conte uma história, eduque, não venda |
| **Problem Aware** | Sabe do problema, não da solução | Agite a dor, então revele que existe solução |
| **Solution Aware** | Sabe que existe solução, não te conhece | Diferencie sua solução das outras |
| **Product Aware** | Te conhece, não está convencido | Prove, depoimentos, garantia, oferta |
| **Most Aware** | Pronto para comprar | Mostre a oferta, facilite a ação |

**Regra de Schwartz:** *"Não crie desejo — conecte ao desejo que já existe."*

## David Ogilvy — Princípios Imortais

1. **Pesquise o produto até saber tudo que há para saber** — o melhor diferencial já está na pesquisa
2. **A headline é 80% da peça** — se ela falha, o resto é irrelevante
3. **O cliente não é idiota — ele é sua esposa/marido** — nunca subestime
4. **Específico sempre vence genérico** — "Rolls-Royce a 60mph, o barulho é o relógio"
5. **Teste, teste, teste** — nunca aposte em opiniões quando pode ter dados
6. **Copy longa vence copy curta** — quando tem algo relevante a dizer

## Alex Hormozi — $100M Offers Framework

### Grand Slam Offer = Dream Outcome × Perceived Likelihood × Speed ÷ Effort & Sacrifice

**Value Stack Obrigatório:**
1. Liste tudo que o cliente recebe (produto + bônus + suporte)
2. Atribua valor honesto a cada item
3. Some o total
4. Mostre o preço real (muito abaixo do total)
5. "Não faz sentido dizer não"

**Bônus que convertem:**
- Resolução de objeções como bônus ("o script para superar X" como bônus)
- Velocidade ("começa em 24h")
- Garantia de resultado ("ou devolvo")

## Dan Kennedy — NIDO Formula + Direct Response

**NIDO:**
- **N**ome do problema específico
- **I**nteresse imediato (por que me importa agora?)
- **D**esejo de solução + deadline
- **O**ferta irresistível com CTA claro

**Kennedy's Rules:**
1. Sempre tenha um deadline (real)
2. Sempre tenha um CTA explícito
3. Sempre faça a oferta pelo menos 3 vezes
4. Sempre teste o preço (não assuma que mais barato vende mais)

## Gary Halbert — Storytelling Direto

**The Boron Letters Principles:**
- "Starving Crowd bate Always" — encontre quem está com fome antes de criar o produto
- Personalize: escreva como se fosse uma carta para UMA pessoa específica
- "AIDA em esteróides": Attention → Interest → Desire → Action (cada linha deve arrastar para a próxima)
- A história vende mais que o argumento

**One-Legged Golfer Framework:**
1. Situação aparentemente impossível (problema dramático)
2. Tentativas que falharam
3. A virada (seu Unique Mechanism)
4. Resultado específico
5. "Se funcionou para ele, funcionará para você"

## Robert Cialdini — 7 Princípios

| Princípio | Aplicação em LP |
|-----------|----------------|
| **Reciprocidade** | Ofereça algo de valor antes de pedir (lead magnet, conteúdo gratuito) |
| **Compromisso** | Micro-CTAs progressivos (email antes de compra, quiz antes de oferta) |
| **Prova Social** | Depoimentos, números ("2.847 clientes"), logos, ratings |
| **Autoridade** | Credenciais, prêmios, imprensa, certificações em evidência |
| **Afinidade** | Tom, visual e exemplos que parecem "para mim" |
| **Escassez** | Vagas/estoque limitado (com motivo real) |
| **Unidade** | "Somos do mesmo grupo" — valores, identidade, pertencimento |

## Claude Hopkins — Scientific Advertising

1. **Reason Why** — sempre explique POR QUE a oferta é boa (não apenas que é boa)
2. **Specificity** — "53 sabores" bate "muitos sabores"
3. **Test Everything** — copy é ciência, não arte
4. **Headlines as Ads** — a headline deve ser um anúncio completo por si só

## Joe Sugarman — Psychological Triggers

Top 10 triggers para landing pages:
1. **Curiosidade** — "O que os ricos sabem sobre dinheiro que nunca te contaram"
2. **Urgência** — deadline real, escassez real
3. **Exclusividade** — "apenas para membros" / "selecionados"
4. **Prova** — dados, estudos, casos reais com números
5. **Antecipação** — construa expectativa antes de revelar o produto
6. **Identificação** — avatar específico na copy ("se você é X...")
7. **Satisfação** — mostre o feeling de ter o problema resolvido
8. **Simplificação** — torne o complexo simples de entender e obter
9. **Segurança** — garantia, HTTPS, CNPJ, depoimentos verificados
10. **Alinhamento de valores** — compartilhe crenças antes de vender
