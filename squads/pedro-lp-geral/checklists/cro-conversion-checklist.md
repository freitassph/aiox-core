# CRO Conversion Checklist

## Above the Fold (3 segundos test)
```
□ Headline: comunica proposta de valor em < 5 segundos?
□ CTA principal: visível sem scroll no mobile E desktop?
□ Social proof mínima: número, rating ou logo above the fold?
□ Visual: imagem/vídeo confirma o que o texto promete?
□ Responde: o que é? para quem? por que agora?
```

## Cialdini's 7 Principles
```
□ Reciprocidade: algo de valor oferecido antes de pedir?
□ Compromisso: micro-conversões progressivas antes do CTA principal?
□ Prova Social: depoimentos, números, logos em evidência?
□ Autoridade: credenciais, prêmios, imprensa bem posicionados?
□ Afinidade: tom e visual claramente "para mim"?
□ Escassez: urgência real e justificada (não fake)?
□ Unidade: "somos do mesmo grupo" transmitido?
```

## Friction
```
□ Formulário: máximo 3 campos no topo (mais na segunda etapa)?
□ CTA: texto específico (não "saiba mais" ou "clique aqui")?
□ Distrações: menus desnecessários, links externos removidos?
□ Velocidade: < 3s no mobile (percepção de confiança)?
□ Mobile CTA: acessível com polegar sem esforço?
```

## Trust Architecture
```
□ HTTPS visível?
□ CNPJ + endereço real no footer?
□ Garantia em destaque (elimina risco)?
□ Depoimentos: foto real + nome completo + contexto?
□ "Borrowed authority": logos conhecidos em evidência?
□ LGPD badge / Política de Privacidade linkada?
```

## Offer Architecture
```
□ Value Stack: valor percebido >> preço pedido?
□ Bônus inesperados aumentam percepção de valor?
□ Garantia: formulação forte que elimina objeção de risco?
□ CTA fala o que acontece DEPOIS do clique?
□ Anti-objeções: as 5 principais respondidas?
```

## Urgency & Closing
```
□ Urgência: real e justificada (não inventada)?
□ Fechamento: resumo da oferta em bullet points?
□ CTA final: o mais específico e irresistível de toda a página?
□ Anti-CTA: consequência da inação mencionada?
```
