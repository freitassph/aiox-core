# LGPD Compliance Checklist

## Obrigatório para toda landing page publicada no Brasil

### Consentimento
```
□ Banner de cookies: aparece no primeiro acesso (antes de pixels carregarem)
□ Categorias distintas: essenciais / analytics / marketing
□ Aceitar todos: botão presente
□ Recusar não essenciais: botão presente e funcional
□ Personalizar: opção granular por categoria
□ Revogação: usuário pode alterar preferências depois
□ Pixels carregam apenas após consentimento das categorias relevantes
```

### Documentação Legal
```
□ Política de Privacidade: link visível no rodapé e no banner de cookies
□ Termos de Uso: link no rodapé (se houver venda ou cadastro)
□ Controlador identificado: nome + CNPJ + endereço
□ DPO (se aplicável): nome + e-mail
□ Dados coletados: listados com finalidade para cada um
□ Retenção de dados: prazo informado
□ Canal DSAR: e-mail ou formulário para solicitações de titulares
```

### Minimização de Dados
```
□ Formulários: apenas campos estritamente necessários
□ Dados sensíveis (saúde, origem étnica, etc.): não coletados sem necessidade
□ Sem campos desnecessários para o objetivo da landing
```

### Técnico
```
□ HTTPS: dados em trânsito sempre criptografados
□ Formulários: dados enviados via POST (nunca via GET/URL)
□ Analytics: IP anonimizado (GA4 configurado com anonimização)
□ CookieYes / Cookiebot / equivalente: configurado e ativo
```
