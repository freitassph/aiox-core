# Task: LP Briefing — Coleta Estratégica Completa

## Objetivo
Coletar todas as informações necessárias para iniciar o projeto com clareza total e identificar o Brand Archetype que guiará design e copy. Sem briefing completo, nenhuma outra task pode começar.

## Agente Responsável
`lp-prd-architect` (Camila Torres)

## Método
Entrevista guiada com o cliente. Pode ser 1 sessão de 60 min ou assíncrono via formulário. Ao final, compilar em documento estruturado e declarar o Brand Archetype.

---

## ETAPA 1 — Brand Archetype Detection (PRIMEIRO)

```
Identificar o archetype antes de tudo:
□ Qual é o segmento de mercado?
□ Qual é a faixa de preço do produto/serviço?
□ 3 marcas que admira e por quê?
□ Tom desejado: sofisticado / energético / caloroso / técnico / divertido?
□ Como o cliente deve se sentir ao ver a landing: 3 palavras-chave

→ DECLARAR: Brand Archetype = [LUXURY / BOLD / HUMAN / TECHNICAL / PLAYFUL]
```

## ETAPA 2 — Negócio & Oferta

```
□ Nome do negócio / marca
□ Produto ou serviço principal (descrever detalhadamente)
□ Preço exato ou faixa
□ Modelo de venda: venda direta / captação de lead / agendamento / outro
□ Principal diferencial competitivo (o que não pode ser copiado?)
□ Unique Mechanism (tem nome próprio para o método/sistema?)
□ Ticket médio atual → ticket desejado após a landing
□ Por que um cliente escolheria você e não o concorrente?
```

## ETAPA 3 — Avatar do Cliente

```
□ Quem compra hoje? (idade, gênero, ocupação, renda estimada)
□ Qual é a dor/problema principal que resolve?
□ O que o cliente REALMENTE quer (transformação, não o produto)?
□ O que ele já tentou antes e não funcionou?
□ Quais são os maiores medos em relação à compra?
□ O que o faria comprar imediatamente?
□ Nível de consciência atual:
  □ Unaware (não sabe que tem o problema)
  □ Problem Aware (sabe do problema, não sabe da solução)
  □ Solution Aware (sabe que existe solução, não conhece você)
  □ Product Aware (te conhece, não está convencido)
  □ Most Aware (pronto, só precisa da oferta certa)
```

## ETAPA 4 — Provas & Credenciais

```
□ Tempo de mercado / anos de experiência
□ Número de clientes / projetos / atendimentos realizados
□ Resultado mais expressivo com número real (ex: "cliente X aumentou 340% em 60 dias")
□ Depoimentos disponíveis: texto / screenshot / vídeo
□ Logos de clientes notáveis / parceiros
□ Imprensa: apareceu em onde?
□ Prêmios, certificações, formações relevantes
```

## ETAPA 5 — Concorrência

```
□ Principais concorrentes (URLs das landing pages deles)
□ O que eles fazem melhor que você?
□ O que você faz melhor que eles?
□ Tem landing page atual? URL + o que não funciona nela?
```

## ETAPA 6 — Objetivos & Métricas

```
□ Objetivo principal: venda / lead / agendamento
□ Meta de conversão esperada: ___%
□ Volume de tráfego esperado por mês
□ Principal fonte de tráfego: orgânico / paid (Google/Meta) / social / email / indicação
□ Prazo de entrega desejado
□ Orçamento disponível (define nível de complexidade)
```

## ETAPA 7 — Técnico & Visual

```
□ Domínio: já existe / vai registrar / domínio atual
□ Hospedagem preferida: Vercel (recomendado) / outra
□ Logo: arquivo disponível / criar / adaptar
□ Identidade visual existente: sim (enviar) / criar do zero
□ Fotos profissionais: disponíveis / banco de imagens / AI images
□ Vídeo: tem / vai produzir / não vai usar
□ Integrações necessárias: CRM / e-mail marketing / WhatsApp / agendamento online / pagamento / outra
□ Prazo de lançamento
```

## Output Esperado

- `briefing-document.md` — Briefing completo estruturado
- `brand-archetype-report.md` — Archetype declarado + racional
- `avatar-profile.md` — Persona com dores, desejos e objeções
- `project-scope.md` — Escopo: seções, features, integrações
- `awareness-strategy.md` — Nível de consciência + estratégia de copy
