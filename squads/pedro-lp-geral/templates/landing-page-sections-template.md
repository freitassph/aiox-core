# Landing Page Sections Template — Estrutura Universal

## Seções Obrigatórias (toda landing page)
1. **Header** — logo + CTA button + mobile menu
2. **Hero** — headline + subheadline + CTA + prova social mínima + visual
3. **Problema** — agitar a dor, normalizar, ampliar, pivot
4. **Solução** — Unique Mechanism + 3 passos + diferenciação
5. **Prova Social** — depoimentos + números + logos
6. **Oferta** — value stack + preço + garantia + CTA
7. **FAQ** — 6-10 perguntas que eliminam objeções
8. **CTA Final** — resumo + urgência + CTA irresistível
9. **Footer** — info legal + LGPD + links úteis

## Seções Opcionais (por objetivo e complexidade)
- **Stats** — números de prova social (clientes, resultados, tempo)
- **Processo** — como funciona em 3-4 passos (reduz ansiedade)
- **Sobre** — founder story / autoridade (para produtos com persona)
- **Comparativo** — você vs. alternativas (posicionamento agressivo)
- **Galeria/Portfolio** — para serviços visuais (design, imóveis, food)
- **Mídia** — aparições em imprensa
- **Contato** — para negócios que preferem WhatsApp/formulário a venda direta

## Copy Placeholders por Seção

### Hero
```
[HEADLINE] — máximo 12 palavras
[SUBHEADLINE] — 1-2 frases, 20-30 palavras
[HERO_CTA] — verbo de ação + resultado específico
[SOCIAL_PROOF_MINI] — "X clientes" ou rating ou logo
[HERO_IMAGE_ALT] — descrição da imagem
```

### Problema
```
[DOR_PRINCIPAL] — a maior frustração do avatar
[DOR_SECUNDARIA_1] — segunda dor relacionada
[DOR_SECUNDARIA_2] — terceira dor relacionada
[NORMALIZACAO] — "você não está sozinho: X% de..."
[AMPLIFICACAO] — "e se continuar assim, em 12 meses..."
[PIVOT] — "mas existe uma forma diferente..."
```

### Solução
```
[UNIQUE_MECHANISM_NOME] — nome proprietário do método
[PASSO_1] — primeiro passo do método
[PASSO_2] — segundo passo do método
[PASSO_3] — terceiro passo do método
[DIFERENCIACAO] — por que outros métodos falham?
```

### Oferta
```
[ITEM_PRINCIPAL] — produto/serviço principal (valor: R$X)
[BONUS_1] — primeiro bônus (valor: R$Y)
[BONUS_2] — segundo bônus (valor: R$Z)
[TOTAL_VALOR] — soma dos valores acima
[PRECO_REAL] — quanto realmente custa
[GARANTIA] — texto da garantia + período
[CTA_OFERTA] — CTA da seção de oferta
```
