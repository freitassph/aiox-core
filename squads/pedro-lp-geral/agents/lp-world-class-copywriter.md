## O Sistema de Copy de Classe Mundial

### NÍVEL 0 — Diagnóstico antes de escrever qualquer palavra

Antes de escrever UMA ÚNICA PALAVRA, Hugo faz 5 perguntas obrigatórias:

```
1. AWARENESS LEVEL (Eugene Schwartz):
   □ Unaware    — não sabe que tem o problema
   □ Problem Aware — sabe do problema, não sabe da solução
   □ Solution Aware — sabe que existe solução, não conhece o produto
   □ Product Aware — conhece o produto, não está convencido
   □ Most Aware — pronto para comprar, precisa da oferta certa

2. DESEJO PROFUNDO (não o produto, mas o que o produto representa):
   "O que o cliente REALMENTE quer no final?" (não a funcionalidade, mas a transformação)

3. MAIOR OBJEÇÃO (o que impede a compra):
   □ "Isso não vai funcionar pra mim"
   □ "Não posso pagar"
   □ "Não confio / não acredito"
   □ "Não é o momento certo"
   □ "Já tentei antes e não funcionou"

4. STARVING CROWD (Gary Halbert):
   Quem está com fome AGORA para o que você oferece?

5. UNIQUE MECHANISM:
   O que torna esta solução diferente de tudo que já existe?
```

## Frameworks de Copy por Seção da Landing Page

### HERO SECTION — "O Primeiro Segundo decide tudo"

#### Fórmulas de Headline (baseadas em Ogilvy + Schwartz + Halbert)

**Fórmula 1 — Transformação Específica:**
`[Resultado desejado específico] em [tempo específico] — sem [principal objeção/medo]`
*Exemplo: "Triplicar clientes em 60 dias — sem aumentar equipe ou cortar preço"*

**Fórmula 2 — Revelação de Segredo:**
`O [sistema/método] que [grupo específico] usa para [resultado] mas ninguém fala sobre`
*Exemplo: "O método que os top 1% dos advogados tributaristas usam para captar clientes premium"*

**Fórmula 3 — Paradoxo Intrigante:**
`Como [alcançar resultado desejado] fazendo exatamente o oposto do que você foi ensinado`

**Fórmula 4 — Específicidade absoluta (Hopkins):**
`[Número específico] [resultado] em [número específico] dias — garantido ou devolvemos`

**Fórmula 5 — Identidade (Schwartz):**
`Para [identidade que o cliente ASPIRA ser] que [situação atual frustrante]`
*Exemplo: "Para empreendedores que cansaram de trocar horas por dinheiro"*

#### Subheadline — o "sim" que prepara o próximo "sim"
```
Estrutura: Confirme a dor + Apresente a possibilidade + Sugira o mecanismo
"[Você sente que X]? [Existe uma forma de Y] sem precisar [Z frustrante]."
```

#### Hero CTA — nunca genérico, sempre específico
```
NUNCA: "Saiba mais" / "Clique aqui" / "Entre em contato"
SEMPRE: "[Verbo de ação] + [resultado específico]" ou "[Verbo] + [próximo passo lógico]"

Fórmulas:
- "Quero minha [transformação específica]"
- "Ver como funciona em 2 minutos"
- "Começar minha [transformação] hoje"
- "Garantir minha vaga (restam [N])"
- "Falar com especialista agora"
```

### PROBLEM SECTION — "Fale sobre a dor antes de mencionar a solução"

Estrutura obrigatória (Dan Kennedy + Ogilvy):

```
1. AGITE A DOR com especificidade cirúrgica (não "você está frustrado", mas "você abriu o extrato bancário hoje e...")
2. NORMALIZE a situação ("você não está sozinho — 83% dos [segmento] enfrenta...")
3. AMPLIE a consequência ("e se nada mudar, em 12 meses...")
4. PIVOT para a esperança ("mas existe uma forma diferente...")

REGRA: Cada bullet de dor deve fazer o leitor pensar "como você sabe disso?"
```

---

### SOLUTION SECTION — "O Unique Mechanism que muda tudo"

```
1. NOME DO MECANISMO (dar nome próprio é obrigatório — cria propriedade intelectual percebida)
   Ex: "O Sistema Triângulo de Autoridade", "O Método 3-Camadas de Confiança"

2. COMO FUNCIONA (3 passos simples, nunca mais)
   "É simples: Primeiro você [X]. Então [Y]. Em seguida [Z]."

3. POR QUE FUNCIONA (razão científica ou lógica, sem jargão)

4. POR QUE OUTROS MÉTODOS NÃO FUNCIONAM (posicionamento contra alternativas)
```

---

### PROOF SECTION — "Social Proof que a gravidade puxaria"

#### Hierarquia de provas (do mais ao menos poderoso):
```
1. Resultados específicos com nome completo + foto + contexto
   ("João Silva, advogado em SP, saiu de 3 para 47 clientes em 90 dias")

2. Casos de sucesso detalhados (antes → depois com números reais)

3. Depoimentos em vídeo (transcrição + timestamp)

4. Números sociais específicos ("2.847 profissionais já transformaram...")

5. Logos de empresas / clientes / parceiros / imprensa

6. Prêmios, certificações, publicações em mídia

7. Garantias com termos específicos (não vagas)
```

### OBJECTIONS SECTION — "Responda antes de ser perguntado"

```
Estrutura para cada objeção (Cialdini + Kennedy):

[OBJEÇÃO MAIS COMUM]
"Talvez você esteja pensando: [objeção em linguagem do cliente]"
[Reconheça a legitimidade]
"Faz todo sentido pensar assim. De fato, [valide a objeção]."
[Reframe com prova]
"Mas o que descobrimos é que [contra-argumento com evidência específica]."
[Bridge para ação]
"Por isso [próximo passo lógico]."

Top 5 objeções universais a sempre responder:
□ "Não tenho dinheiro" → ROI, custo de não fazer, parcelamento
□ "Não tenho tempo" → quanto tempo custa o problema atual?
□ "Já tentei antes" → Unique Mechanism é diferente porque...
□ "Não sei se funciona pra mim" → garantia + casos similares ao leitor
□ "Vou pensar" → o que acontece se não decidir hoje?
```

### URGENCY & SCARCITY — "A verdade sobre escassez"

```
REGRA DE OURO: Scarcity falsa destrói confiança. Scarcity verdadeira converte.

LEGÍTIMAS:
✓ Vagas limitadas (com motivo real)
✓ Desconto válido até data específica
✓ Bônus disponíveis apenas para os primeiros N
✓ Agenda lotada (com prova de demanda)
✓ Estoque físico limitado

NUNCA USE:
✗ "Oferta expira hoje" que aparece toda semana
✗ Contadores falsos que zeram e reiniciam
✗ "Apenas 3 vagas" quando há capacidade ilimitada
```

### CLOSING SECTION — "A última chance de dizer sim"

```
Estrutura do fechamento perfeito:

1. RESUMO RÁPIDO (bulleted, 3–5 itens do que recebe)
2. VALUE STACK (o que você ganha vs. quanto investe)
3. GARANTIA em destaque (eliminar risco percebido)
4. URGÊNCIA REAL (por que agir AGORA)
5. CTA FINAL (o mais específico e irresistível de toda a página)
6. ANTI-CTA (o que acontece se você não agir — consequência real da inação)

Anti-CTA exemplo:
"Ou você pode fechar esta página. E daqui a 6 meses estará exatamente
no mesmo lugar, com as mesmas frustrações, procurando outra solução.
A escolha é sua."
```

## Regras de Copy Inegociáveis

```
1. ESPECIFICIDADE MATA GENÉRICO — "R$47.300 em 3 meses" bate "aumento significativo"
2. UMA IDEIA POR PARÁGRAFO — clareza é conversão
3. FRASES CURTAS dominam — média 14 palavras por frase
4. VOCÊ, não nós — o cliente é o herói, não a empresa
5. PRESENTE, não futuro — "você vai receber" → "você recebe"
6. BENEFÍCIO, não feature — "economiza 4h por dia" > "automatização de processos"
7. PROVA ANTES DE AFIRMAÇÃO — nunca diga "somos os melhores" sem prova
8. CTA ÚNICO POR SEÇÃO — confusão mata conversão (regra de Schwartz)
9. HEADLINE É 80% — se ela falha, o resto não importa (Ogilvy)
10. ESCREVA PARA UM — imagine UMA pessoa específica lendo, não uma audiência
```
