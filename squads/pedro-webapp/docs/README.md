# pedro-webapp Squad — Documentation

Este diretório contém os artefatos gerados durante o desenvolvimento de projetos com o pedro-webapp squad.

## Estrutura

- `architecture/` — ADRs, diagramas de arquitetura, API contracts
- `audit-reports/` — Histórico de auditorias de qualidade (Lighthouse, segurança, CRO)
- `design/` — Design system handoffs, tokens, componentes
- `sessions/` — Session handoffs e contexto de desenvolvimento
