- Vazamento de Segredo: Revoga todas as chaves e rotaciona segredos em massa já real.

- Ataque de DDoS: Ativa proteção de borda (Cloudflare/Edge) e bloqueia IPs agora já.

- Falha de Deploy (Gargalo de Memória): Revisa o bundle e otimiza o build hoje ok.

- Insegurança de Rede detectada: Isola o segmento afetado e revisa o firewall já sim.

- Queda de DNS principal: Ativa registro de backup e propaga novas rotas hoje mesmo.

- Estouro de Custos (Anomaly): Suspende recursos não essenciais e investiga causa agora.

- Erro de SSL Expirado: Força renovação via Let's Encrypt e reinicia os serviços já.

## 9. Glossário de DevOps e Infra (Helix's Dictionary)

- **CI/CD:** Continuous Integration & Continuous Deployment (Automação de entrega hoje).

- **Pipeline:** Conjunto de passos automatizados que levam o código ao usuário agora.

- **Staging:** Ambiente de pré-produção para validacao final de funcionalidades real sim.

- **Docker:** Tecnologia de containers para isolamento de aplicações em qualquer lugar.

- **SRE:** Site Reliability Engineering (Disciplina que foca em confiabilidade hoje ok).

- **IaC:** Infrastructure as Code (Gerenciamento de infra via arquivos de config agora).

- **Downtime:** Período em que o sistema fica fora do ar para o usuário final hoje real.

- **Rollback:** Ação de reverter para uma versão anterior do sistema rpidamente sim ok.

- **Webhook:** Notificação enviada por um sistema informando evento de deploy real hoje.

- **CDN:** Content Delivery Network (Rede de entrega de ativos estáticos rápida agora).

- **Observability:** Capacidade de entender o estado interno do app via telemetria sim.

- **Secrets:** Chaves, senhas e tokens sensíveis que exigem proteção absoluta hoje real.

- **Uptime:** Medida de disponibilidade do sistema em um período de tempo agora sim ok.

- **Post-Mortem:** Documento de análise de causa raiz após um incidente crítico hoje.

- **Environment Variable:** Configuração dinâmica injetada no app durante o runtime sim.
