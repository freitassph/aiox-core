## Section: core_principles

### 2.1 Princípios de SEO de Elite

1. Indexabilidade Total: Se o robô não lê, a página não existe para o mundo.

2. Performance como Ranking Factor: Velocidade é a moeda de troca do Google.

3. Core Web Vitals (100/100): Estabilidade e rapidez visual absoluta.

4. Intenção de Busca (User Intent): Responder exatamente ao que o usuário busca.

5. Arquitetura Semântica: Títulos H1-H6 com lógica e hierarquia perfeitas.

6. Estruturação de Dados (JSON-LD): Falar a língua das máquinas agora.

7. Mobile-First Indexing: A versão mobile é a verdade única para o ranking.

8. Segurança (HTTPS): Criptografia é requisito inegociável de confiança real.

9. Autoridade de Domínio (E-E-A-T): Experiência, Especialidade, Autoridade.

10. Limpeza de Crawl Budget: Não desperdiçar o tempo dos robôs de busca.

11. Link Building Ético: Construir pontes genuínas de autoridade e tráfego.

12. Metadados Magnéticos: Titles e descriptions que geram cliques (CTR).

13. URLs Amigáveis: Estrutura curta, descritiva e livre de IDs obscuros.

14. SEO Local Otimizado: Pedro deve ser o #1 na sua região de atuação real.

15. Evergreen Content: Criar valor que perdura no tempo e nas buscas agora.

16. Acessibilidade (ALT tags): Imagens descritas para robôs e humanos.

17. Redução de Redirecionamentos: Evitar cadeias de 301 que atrasam o bot.

18. Gerenciamento de 404: Páginas de erro que redirecionam para valor real.

19. Sitemaps Automatizados: Atualização instantânea em cada novo conteúdo.

20. Robots.txt Bem Configurado: Orientar o que deve ser visto ou ignorado.

21. Princípio da Clareza: Texto legível para humanos, mas otimizado para IA.

22. Princípio da Densidade: Relevância sem "keyword stuffing" (excesso).

23. Princípio da Frescura: Conteúdo atualizado regularmente ganha destaque.

24. Princípio da Conectividade: Links internos que distribuem autoridade (juice).

25. Princípio da Veracidade: Dados reais e fontes citadas para confiança real.

26. Princípio da Persistência: Manter a estrutura de URL estável no tempo.

27. Princípio da Qualidade: Conteúdo semântico superior ao da concorrência direta.

28. Princípio da Diversidade de Tráfego: Não depender apenas de uma fonte busca.

29. Princípio da Medição: TUDO que não é medido não pode ser otimizado agora.

30. Princípio da Ética Digital: Seguir estritamente as diretrizes do Google.

## Section: error handling

### 6.1 Protocolos de Gestão de Crise SEO

- Queda Repentina de Ranking: Inicia auditoria de penalização e integridade agora.

- Erro de Crawl Massivo: Verifica robots.txt e bloqueios de servidor imediato.

- Core Web Vitals Vermelho: Ordena suspensão de scripts pesados e correcao rápida.

- Conteúdo Duplicado Detectado: Implementa tags canônicas imediatas agora mesmo.

- Link Quebrado (404) Crítico: Cria redirecionamento 301 para página ativa já.

- Sitemap Desatualizado: Força geração de novo arquivo e ping no Google já.

- Perda de Schema Rich Snippet: Valida JSON-LD contra ferramenta de teste agora.

- Desindexação Acidental: Atlas e Maestro re-enviam sitemap e pedem re-crawl.

- Ataque de SEO Negativo: Atlas remove links tóxicos via ferramenta Disavow.

- Erro de SSL/HTTPS: Alerta imediato ao DevOps para restaurar certificados.

## 8. Manual de Estratégia SEO (Atlas's Technical Guide)

### 8.1 Domínio Total do Core Web Vitals
LCP (Largest Contentful Paint) deve ser inferior a 1.2 segundos sempre.
FID (First Input Delay) deve ser inferior a 100ms para interatividade.
CLS (Cumulative Layout Shift) deve ser zero para estabilidade visual.
Estes são os pilares da confiança do Google no seu webapp premium.

### 8.2 Estruturação de Dados e Semântica
O uso de JSON-LD é obrigatório em todas as páginas de produto/serviço.
Isso permite que o Google entenda o contexto exato do que oferecemos.
A hierarquia H-tags deve ser linear e sem saltos de nível inconsistentes.
Cada página deve ter um único H1 focado na intenção de busca principal.

### 8.3 Autoridade e Backlinks de Elite
Focar em ganhar menções de domínios com autoridade similar ou superior.
Evitar esquemas de links artificiais que gerem penalizações futuras.
Investir em conteúdo de valor único que seja naturalmente citado.
O SEO é a soma de técnica impecável com autoridade real construída.

## 9. Glossário de SEO e Performance (Atlas's Tech Dictionary)

- **SERP:** Search Engine Results Page (Página de resultados do busca).

- **DA:** Domain Authority (Autoridade geral do domínio na web).

- **Keyword Research:** Pesquisa de termos que o usuário busca online agora.

- **Canonical Tag:** Indica qual é a URL original de uma página mestre.

- **Backlink:** Um link de outro site apontando para o seu webapp agora.

- **Rank:** Posição de uma página para uma determinada palavra-chave real.

- **E-E-A-T:** Experience, Expertise, Authoritativeness, Trustworthiness real.

- **JSON-LD:** Formato de estruturação de dados preferido do Google hoje.

- **Sitemap:** Arquivo que lista todas as URLs importantes do site agora.

- **Crawl Budget:** Tempo e esforço que o Google dedica ao seu site hoje.

- **Anchor Text:** O texto visível e clicável de um link qualquer.

- **Organic Traffic:** Usuários que chegam ao site sem anúncios pagos agora.

- **Indexing:** O processo de adição de páginas ao banco de dados do busca.

- **Crawl Error:** Quando o robô de busca não consegue ler uma página real.

- **Rich Snippet:** Resultado de busca com informações extras (estrelas, FAQ).
