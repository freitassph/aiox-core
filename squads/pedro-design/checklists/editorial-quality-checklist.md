# Editorial Quality Checklist

> Checklist para ebooks, PDFs, relatórios e materiais editoriais
> Executor: pdg-auditor (Sentinel)

## Sistema Editorial

- [ ] Grid definido e consistentemente aplicado
- [ ] Tipografia com hierarquia clara (H1 > H2 > H3 > body > caption)
- [ ] Escala tipográfica respeitada (tamanhos do sistema)
- [ ] Line-height adequado (1.5x+ para body text)
- [ ] Paleta de cores consistente com brand system

## Capa

- [ ] Transmite claramente o tema/valor do conteúdo
- [ ] Título legível e impactante
- [ ] Subtítulo complementa o título
- [ ] Logo/branding presente e correto
- [ ] Imagem/visual de alta qualidade

## Conteúdo e Hierarquia

- [ ] Índice presente e funcional
- [ ] Títulos de capítulo claramente diferenciados
- [ ] Body text com tamanho mínimo de 11pt (print) / 16px (digital)
- [ ] Callouts e highlights estilizados de forma consistente
- [ ] Pull quotes estilizados (se houver)

## Imagens e Visuais

- [ ] Imagens em alta resolução (300 DPI para print)
- [ ] Captions presentes em todas as imagens
- [ ] Imagens alinhadas ao grid
- [ ] Alt text / descrições para acessibilidade (digital)

## Ritmo Visual

- [ ] Alternância de páginas densas e visuais
- [ ] Espaço em branco utilizado intencionalmente
- [ ] Páginas abertura de capítulo distintas das páginas de conteúdo
- [ ] CTA final presente

## Técnico (Print)

- [ ] Sangria 3mm em todos os lados
- [ ] Zona segura 5mm respeitada
- [ ] Fontes embedded
- [ ] Imagens em CMYK (se print)
- [ ] Resolução 300 DPI mínimo

## Técnico (Digital)

- [ ] Links do TOC funcionando
- [ ] Links externos corretos
- [ ] RGB (não CMYK)
- [ ] Bookmarks/outline do PDF configurados
- [ ] Tamanho do arquivo adequado (<10MB para email)
