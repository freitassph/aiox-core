# Mockup Quality Checklist

> Checklist de qualidade para todos os mockups do pedro-design
> Executor: pdg-auditor (Sentinel)

## Pre-Render

- [ ] Tipo de mockup definido e adequado ao uso
- [ ] Composição planejada antes da execução
- [ ] Câmera/ângulo definido
- [ ] Iluminação especificada
- [ ] Materiais PBR configurados

## Design Application

- [ ] Design aplicado corretamente (sem distorção)
- [ ] Perspectiva do design está correta para o ângulo da câmera
- [ ] Cores do design preservadas (nenhuma alteração por iluminação)
- [ ] Logotipos/texto legíveis
- [ ] Elementos de design alinhados (não tortos)

## Realismo

- [ ] Sombras condizentes com a fonte de luz
- [ ] Materiais têm textura convincente
- [ ] Reflexos são naturais (não artificiais demais)
- [ ] Produto tem presença física (não "flutuando")
- [ ] Ambiente/contexto é crível

## Qualidade de Render

- [ ] Resolução mínima conforme especificado (HD/2K/4K/8K)
- [ ] Sem artefatos visíveis (pixelação, aliasing, noise excessivo)
- [ ] Foco correto (produto em foco quando deve estar)
- [ ] Depth of field adequado para o contexto
- [ ] Color grading consistente com o tom desejado

## Formato e Entrega

- [ ] Arquivo nomeado corretamente
- [ ] Formato correto (JPEG, PNG, TIFF conforme uso)
- [ ] Tamanho de arquivo razoável para o contexto
- [ ] Variações entregues (dark/light/lifestyle conforme solicitado)
- [ ] Fundo transparente onde necessário (PNG)
