## 3. Melhores Práticas por Formato

### Posts de Feed (Instagram/Facebook/LinkedIn)

| Prática | Recomendação |
|---|---|
| **Proporção ideal** | 4:5 (retrato) — ocupa mais tela |
| **Texto na imagem** | Máximo 20% da área — manter limpo |
| **Branding** | Logo sutil no canto ou estilo consistente |
| **Cores** | Paleta consistente com a marca |
| **Tipografia** | Máximo 2 fontes, legível em mobile |
| **Frequência** | Instagram: 3-5x/semana, Facebook: 1-2x/semana, LinkedIn: 2-5x/semana |
| **Horário ideal** | Terça a quinta, 9h-12h (varia por audiência) |

### Stories (Instagram/Facebook/WhatsApp)

| Prática | Recomendação |
|---|---|
| **Duração por story** | 5-15 segundos (antes de pular) |
| **Número por dia** | 3-7 stories |
| **Zona segura** | 250px top e bottom livres de texto |
| **Interatividade** | Usar polls, questions, sliders (aumenta engajamento) |
| **Sequência** | Primeiro story deve prender atenção |
| **CTA** | Um CTA claro por story |
| **Consistência visual** | Templates para manter identidade |

### Reels / TikTok / Shorts (Vídeo Curto)

| Prática | Recomendação |
|---|---|
| **Duração ideal** | 7-15 segundos (melhor completion) |
| **Hook** | Primeiros 2 segundos são críticos |
| **Legendas** | Sempre — 70% assistem sem som inicialmente |
| **Formato** | 9:16 vertical, preencher tela inteira |
| **Áudio** | Usar trending sounds (aumenta alcance) |
| **CTA** | No final ou como texto na tela |
| **Posting** | TikTok: 1-3x/dia, Reels: 3-5x/semana, Shorts: 1-2x/dia |
| **Conteúdo** | Educational, entertaining, ou emotional |

### Carousels / Documentos PDF

| Prática | Recomendação |
|---|---|
| **Número de slides** | 5-10 (Instagram), 5-15 (LinkedIn) |
| **Primeiro slide** | Headline forte, visual impactante |
| **Último slide** | CTA claro + resumo |
| **Design** | Um ponto por slide, texto mínimo |
| **Numeração** | "1/7" para incentivar swipe |
| **Formato** | 1080x1350 (4:5) para Instagram, PDF para LinkedIn |

### Thumbnails de YouTube

| Prática | Recomendação |
|---|---|
| **Dimensões** | 1280 x 720px mínimo |
| **Texto** | Máximo 3-5 palavras, grande e legível |
| **Rosto** | Expressão emocional forte (surpresa, choque) |
| **Contraste** | Alto contraste com o fundo do YouTube |
| **Branding** | Estilo consistente entre todos os vídeos |
| **Cores** | Vibrantes e contrastantes |
| **Testar** | Verificar em tamanho mobile (como aparecerá) |

## 5. Guia de Cores e Design por Plataforma

### Como Cada Plataforma Trata Cores

| Plataforma | Comportamento de Cor | Dica de Design |
|---|---|---|
| **Instagram** | Compressão moderada, satura cores | Usar cores vibrantes, testar após upload |
| **TikTok** | Auto-enhancement, contraste alto | Alto contraste funciona melhor |
| **LinkedIn** | Compressão agressiva | Evitar gradientes sutis, usar cores sólidas |
| **YouTube** | Boa fidelidade de cor | Thumbnails com cores fortes e contrastantes |
| **Facebook** | Compressão similar ao Instagram | Mesma estratégia do Instagram |
| **Pinterest** | Imagens de alta qualidade | Fotos nítidas e bem iluminadas |

### Template Mental de Design para Social Media

| Elemento | Recomendação |
|---|---|
| **Hierarquia** | 1 headline, 1 visual focus, 1 CTA |
| **Espaço negativo** | 30-40% da composição |
| **Texto na imagem** | Mínimo possível — caption complementa |
| **Logo** | Discreto, canto inferior ou superior |
| **CTA visual** | Seta, círculo, ou destaque no elemento clicável |
| **Consistência** | Mesma paleta, tipografia e estilo em todos os posts |

## 8. Métricas-Chave para Monitorar

| Métrica | O que Mede | Benchmark Bom |
|---|---|---|
| **Engagement Rate** | (Likes + Comments + Shares) / Followers | > 2% Instagram, > 4% TikTok |
| **Reach Rate** | Alcance / Seguidores | > 20% orgânico |
| **Completion Rate** | % que assistiu até o final | > 50% (vídeos curtos) |
| **Save Rate** | Saves / Alcance | > 3% indica conteúdo valioso |
| **Share Rate** | Shares / Alcance | > 2% indica conteúdo viral potencial |
| **Click-Through Rate** | Cliques / Impressões | > 1% social, > 3% stories com link |
| **Follower Growth** | Novos seguidores / período | > 3%/mês orgânico |
| **Story Completion** | % que viu todos os stories | > 60% é bom |
