## 1. Fundamentos de Tipografia

### Classificação de Fontes

| Categoria | Características | Exemplos Clássicos | Uso Ideal |
|---|---|---|---|
| **Serif (Serifada)** | Traços decorativos nas extremidades; transmite tradição, elegância | Garamond, Baskerville, Times New Roman, Georgia | Body text longo, editorial, luxury brands |
| **Sans-Serif (Sem serifa)** | Linhas limpas sem decoração; transmite modernidade, clareza | Helvetica, Arial, Inter, Roboto | UI, web body text, tech brands |
| **Display** | Fontes decorativas para tamanhos grandes; personalidade forte | Playfair Display, Bebas Neue, Oswald | Headlines, posters, branding |
| **Monospace** | Largura fixa por caractere; aspecto técnico | Fira Code, JetBrains Mono, Roboto Mono | Código, dados técnicos, terminal |
| **Slab Serif** | Serifas grossas e bloqueadas; força e impacto | Rockwell, Roboto Slab, Josefin Slab | Headlines impactantes, industrial |
| **Handwritten/Script** | Simula escrita manual; pessoal e orgânico | Dancing Script, Caveat, Pacifico | Convites, branding artesanal, pessoal |

### Anatomia Tipográfica

| Termo | Definição |
|---|---|
| **X-Height** | Altura das letras minúsculas (sem ascendentes/descendentes) |
| **Ascender** | Parte da letra que sobe acima do x-height (ex: h, l, b) |
| **Descender** | Parte da letra que desce abaixo da linha de base (ex: g, p, q) |
| **Cap Height** | Altura das letras maiúsculas |
| **Baseline** | Linha imaginária onde as letras se apoiam |
| **Tracking** | Espaçamento geral entre todos os caracteres |
| **Kerning** | Espaçamento ajustado entre pares específicos de letras |
| **Leading** | Espaçamento vertical entre linhas de texto |

### Medidas e Unidades Tipográficas

| Unidade | Descrição | Uso |
|---|---|---|
| **px** | Pixels — unidade fixa | Tamanho base de fontes |
| **em** | Relativo ao font-size do elemento | Padding, margins tipográficas |
| **rem** | Relativo ao font-size do root (html) | Font sizes responsivos |
| **vw/vh** | Relativo à viewport | Tipografia fluida |
| **pt** | Pontos — usado em impressão | Design gráfico impresso |

## 2. Princípios de Font Pairing

### Regras Fundamentais

1. **Contraste com harmonia** — Combine fontes diferentes mas compatíveis
2. **Superfamílias** — Use variações da mesma família (Regular, Bold, Italic, Condensed)
3. **Limite a 2-3 fontes** — Uma para headings, uma para body, opcionalmente uma display
4. **Hierarquia clara** — Heading deve ser visivelmente distinto do body
5. **Mesmo x-height** — Fontes com x-height similar harmonizam melhor
6. **Personalidade complementar** — Uma expressiva + uma neutra funcionam bem

### Estratégias de Combinação

| Estratégia | Descrição | Exemplo |
|---|---|---|
| **Serif + Sans-Serif** | Clássico e versátil; serif no heading, sans no body | Playfair Display + Source Sans 3 |
| **Sans-Serif + Serif** | Moderno; sans no heading, serif no body | Montserrat + Merriweather |
| **Display + Sans-Serif** | Impacto visual + legibilidade | Bebas Neue + Inter |
| **Mesma família** | Harmonia garantida; varia de peso | Roboto (heading bold) + Roboto (body regular) |
| **Superfamília** | Família projetada para funcionar junto | Roboto + Roboto Slab |

## 4. Escalas Tipográficas Responsivas

### O que é uma Escala Tipográfica?

Uma escala tipográfica define tamanhos harmoniosos para diferentes níveis de texto, baseada em uma razão matemática (como a proporção áurea).

### Razões Tipográficas Comuns

| Razão | Nome | Proporção | Quando Usar |
|---|---|---|---|
| **1.125** | Minor Second | 1:1.125 | Interfaces compactas, dashboards densos |
| **1.200** | Major Second | 1:1.200 | Sites padrão, boa maioria dos projetos |
| **1.250** | Minor Third | 1:1.250 | Sites com hierarchy forte, blogs |
| **1.333** | Major Third | 1:1.333 | Landing pages, sites com bastante texto |
| **1.414** | Augmented Fourth | 1:1.414 | Portfólios, sites com headlines grandes |
| **1.500** | Perfect Fifth | 1:1.500 | Landing pages hero, sites promocionais |
| **1.618** | Golden Ratio | 1:1.618 | Designs dramáticos, posters, hero sections |

|---|---|---|

## 5. Tipografia Responsiva com CSS Clamp

### Fórmula

```css
font-size: clamp(mínimo, preferencial, máximo);
```

### Valores Recomendados

| Elemento | Mínimo | Preferencial | Máximo |
|---|---|---|---|
| H1 | 2rem | 5vw | 4rem |
| H2 | 1.75rem | 4vw | 3rem |
| H3 | 1.5rem | 3vw | 2.25rem |
| Body | 1rem | 1.5vw | 1.25rem |
| Small | 0.75rem | 1vw | 0.875rem |

### Exemplo Prático

```css
h1 {
  font-size: clamp(2rem, 5vw, 4rem);
  line-height: 1.1;
  font-weight: 700;
}

body {
  font-size: clamp(1rem, 1.5vw, 1.125rem);
  line-height: 1.6;
}
```

## 8. Line Height e Letter Spacing — Guidelines

### Line Height (Leading)

| Tipo de Texto | Line Height | Justificativa |
|---|---|---|
| Headlines (grande) | 1.0 - 1.2 | Linhas curtas, pouco espaço entre linhas |
| Subtítulos | 1.2 - 1.3 | Equilíbrio entre compacto e legível |
| Body text | 1.5 - 1.7 | Legibilidade ideal para leitura longa |
| Caption/small | 1.4 - 1.5 | Texto pequeno precisa de mais respiro |
| UI labels | 1.0 - 1.2 | Texto curto e funcional |

### Letter Spacing

| Tipo de Texto | Letter Spacing | Justificativa |
|---|---|---|
| All caps heading | 0.05em - 0.15em | Maiúsculas precisam de mais espaço |
| All caps body | 0.05em - 0.1em | Legibilidade em uppercase |
| Body normal | 0 (padrão) | Fontes já vêm otimizada |
| Small text | 0.02em - 0.05em | Texto pequeno ganha com espaçamento extra |
| Display grande | -0.02em - 0 | Letras grandes podem ter tracking negativo |

## 11. Boas Práticas de Performance Tipográfica

| Prática | Descrição | Impacto |
|---|---|---|
| **Limitar pesos** | Usar no máximo 3-4 pesos por família | Reduz ~40-60KB por peso |
| **Preload** | `<link rel="preload">` para font critical | Elimina FOUT/FOIT |
| **font-display: swap** | Mostra fallback até carregar | Melhora LCP |
| **Subsetting** | Incluir apenas caracteres necessários | Reduz 50-80% do tamanho |
| **Variable fonts** | Um arquivo, múltiplos pesos | Reduz requisições HTTP |
| **System fonts** | Usar stack nativo quando possível | Zero download |
