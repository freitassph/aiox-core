## Troubleshooting

### Agents falhando ao ativar

1. **Verificar se o arquivo existe:** `ls agents/pdg-<agent>.md`
2. **Validar YAML header do agent:** O arquivo deve comecar com `---` frontmatter valido contendo `agent:`, `persona:`, `role:`, `squad:`
3. **Verificar sintaxe do agent:** `head -5 agents/pdg-<agent>.md` para confirmar frontmatter
4. **Se o agent corrompeu:** Restaurar a partir do version control: `git checkout agents/pdg-<agent>.md`

### Handoffs quebrados entre agents

1. **Verificar se ambos os agents existem:** `ls agents/pdg-<origin>.md agents/pdg-<target>.md`
2. **Confirmar que o workflow referencia os agents corretos:** `grep -l "pdg-<agent>" workflows/*.yaml`
3. **Verificar compatibilidade de entrada/saida:** A task de saida do agent A deve gerar o formato esperado pela task de entrada do agent B
4. **Executar workflow de auditoria:** `/pedro-design:agents:pdg-auditor` → `*audit-design` para identificar inconsistencias

### Verificar integridade do squad

1. **Validar squad.yaml:** `cat squad.yaml | python3 -c "import sys,yaml; yaml.safe_load(sys.stdin)"` (deve parsear sem erro)
2. **Contar agents:** `ls agents/*.md | wc -l` — deve retornar 23
3. **Contar tasks:** `ls tasks/*.md | wc -l` — deve retornar 29
4. **Verificar orphans:** Comparar arquivos em `agents/`, `tasks/`, `workflows/`, `checklists/`, `templates/` com as listas em `squad.yaml`
5. **Validar workflows:** Cada workflow deve ter `pre_conditions`, `post_conditions` e `error_handling`

### Comandos de diagnostico

```bash
# Listar todos os agents e confirmar count
ls agents/*.md | wc -l

# Verificar qual agent nao tem persona definida
grep -L "persona:" agents/*.md

# Verificar tasks sem checklist
grep -L "checklist:" tasks/*.md

# Verificar workflows sem error handling
grep -L "error_handling:" workflows/*.yaml

# Validar estrutura do squad.yaml
python3 -c "import yaml; yaml.safe_load(open('squad.yaml'))"

# Verificar referencias cruzadas quebradas no squad.yaml
grep "\.md" squad.yaml | grep -v "^#" | while read -r line; do
  file=$(echo "$line" | grep -o '[a-z_-]*\.md')
  [ -n "$file" ] && [ ! -f "agents/$file" ] && [ ! -f "tasks/$file" ] && echo "MISSING: $file"
done

# Contar linhas de cada agent (minimo: 500)
wc -l agents/*.md | sort -n
```
