# Task: Design Unboxing Experience

## Objetivo
Projetar a experiência completa de unboxing de um produto — desde o primeiro contato com a caixa externa até o momento de reveal do produto. O design deve criar uma narrativa emocional progressiva, reforçar o posicionamento da marca e gerar momentos memoráveis que incentivem o compartilhamento orgânico.

## Processo de Execução

### Phase 1 — Mapeamento da Jornada
1. `{*map-current-journey}` — Analisar customer journey fornecida e identificar momentos-chave do unboxing
2. `{*define-emotional-arc}` — Definir arco emocional desejado: curiosidade → antecipação → surpresa → deleite
3. `{*identify-touchpoints}` — Identificar todos os touchpoints físicos: outer box, tissue, insert cards, product holder, seals
4. `{*benchmark-references}` — Benchmark de 10 experiências de unboxing de referência no setor e adjacentes

Entregável: Unboxing Journey Map

### Phase 2 — Sistema de Embalagem
1. `{*design-outer-box}` — Design da caixa externa: forma, abertura, primeira impressão visual
2. `{*design-opening-mechanism}` — Mecanismo de abertura: magnetic flap, ribbon pull, sleeve, tear strip
3. `{*design-inner-layout}` — Layout interno: como os componentes estão organizados e revelados sequencialmente
4. `{*design-product-housing}` — Housing/insert do produto: proteção + apresentação
5. `{*design-secondary-elements}` — Elementos secundários: thank you card, instructions, warranty card, stickers, freebies

Entregável: Packaging System Design

### Phase 3 — Narrativa Visual e Textual
1. `{*craft-copy-sequence}` — Escrever microcopy para cada momento: exterior, abertura, interior, reveal
2. `{*define-visual-reveal}` — Definir elementos de reveal: mensagem oculta que aparece ao abrir, camadas progressivas
3. `{*apply-brand-language}` — Aplicar brand guidelines em cada elemento com consistência
4. `{*design-insert-cards}` — Criar cards de inserção: welcome card, product story, care instructions
5. `{*add-delight-moments}` — Criar pelo menos 2 momentos de surpresa/delight intencionais

Entregável: Narrative + Copy Package

### Phase 4 — Especificação de Materiais
1. `{*spec-box-material}` — Especificar material da caixa: rigid box, corrugated, kraft, com acabamento
2. `{*spec-insert-material}` — Especificar materiais dos inserts: EVA foam, molded pulp, cardboard, foam
3. `{*spec-surface-finishes}` — Especificar acabamentos de superfície: matte lamination, soft-touch, spot UV, foil
4. `{*spec-tissue-paper}` — Especificar papel de seda: cor, branding (printed tissue?), peso
5. `{*spec-inks-adhesives}` — Especificar tintas e adesivos: soy-based, recyclable, branded tape

Entregável: Material Specification Document

### Phase 5 — Protótipos de Experiência
1. `{*create-flat-prototype}` — Criar protótipo plano (dieline) para validação de dimensões
2. `{*create-digital-mockup}` — Criar mockup digital da sequência de abertura
3. `{*create-experience-storyboard}` — Criar storyboard visual da experiência passo-a-passo
4. `{*simulate-unboxing}` — Simular fluxo de unboxing completo com timing e revelações
5. `{*capture-hero-shot}` — Definir o "hero shot" — a foto perfeita para compartilhamento social

Entregável: Experience Prototypes

### Phase 6 — Validação e Handoff
1. Verificar proteção do produto durante transporte (drop test considerations)
2. Validar custos de materiais contra orçamento estimado
3. Confirmar sustentabilidade: materiais recicláveis, mínimo plástico
4. Validar experiência com journey map original
5. Compilar especificações completas para produção

Entregável: Unboxing Experience Package (consolidado)
